Cookies:

Cookies are text files with small pieces of data. It can be username, password or other authentication type tokens. It is used to identify a user from a computer to improve browing experience. It is created by the server upon the username or password given to the server. Data in cookies labeled with an unique ID by which server knows what information specificially to be served.
